## 커스텀 훅의 규칙

- 'use'로 시작하기
- 최상위 수준에서만 훅 호출하기
- 리액트 함수 안에서만 훅 호출하기

### 최상위 수준에서만 훅 호출하기

- 안 좋은 예시

```js
if (조건) {
  useEffect(() => {
    //효과 수행
  }, [dep1, dep2]);
}
```

- 좋은 예시

```js

useEffect(() => {
    if(조건){
        //효과 수행
    }
}, [dep1, dep2])
```

### 리액트 함수 안에서만 훅 호출하기

- 컴포넌트에서 훅을 호출하기
- 커스텀 훅에서 훅을 호출하기

단, 일반적인 자바스크립트 함수에서는 훅을 호출하 말것
